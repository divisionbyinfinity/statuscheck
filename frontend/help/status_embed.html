Inserting Status Badges to your HTML

Overview

You can use the included script, status_embed.js, to add a status badge to any website to return the status of a device, including servers. The device and/or server DOES NOT have to be registered as device in the Status Checker application: it will work for any device or server. You can create your own bage but you will want to look at the status_embed.js script to understand why certain elements needs to be included (basically, the classes ''present'', ''checkme'', and the button must be inside of ''checkme''.

As long as the script and the server are accessible to the the server hosting the webpage - it will work.

Code Example

Below is the code snippet used to create the first example, in the section below:


<div class="status-box present">

   <div class="icon">🔍</div>

   <div class="child-box-left checkme"><button>Ping</button></div>

   <div class="child-box-right"><a href="https://microsoft.com">microsoft.com</a></div>

</div>

Make sure the reference the script, as shown below, before the </body> 


<script src="./status_embded.js"></script>

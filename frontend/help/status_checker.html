Status Checker

Monitor the Status of Networked Devices

Overview

The Status Checker application attempts to ping registered devices at five minute intervals, and if the device does not respond after five minmutes, an email notification can be sent. There is a dashboard on the initial screen, main dashboard, that shows the current status of a each device that is registered in the application. You can add, edit and remove devices, and disable or snooze the email notifications from the Status Check Settings dashboard.

Main Dashboard

On the main dashboard you will see a table that shows all of the registered devices, along with relevant information and the current status of each. If a device is not responding (line is shown in red) or having trouble communicating with the server (line is shown in blue) you can use the PING buttom at the beginning of the line to refresh the status. If you want to connect to the device, if using SSH you will need to configure the device to allow SSH, you can use the action button at the end of the line. The connection options are SSH and HTTPS: you specify the action type when registering the device.

  • Filter Devices

    • At the top of the screen there is a Filter Devices menu that allows you to sort filter by Area and/or Department.

Status Check Settings: Accessing the Administrator Features

To add, edit, or remove devices, or disbale or snooze the email notifications, click on the button with the image shown below to access the Adminsitrator Features.

You will be prompted for a password (these features are managed by your IT staff) and it will launch the Adminsitrator Features dashboard. The Adminsitrator Features dashboard shows the registered devices in a table, very similar to the table on the main dashboard, and the administrator features are available at the end of the line under Admin.

Administrator Features

  • Registering A New Device

    • At the top of the bar, next to Admin you will see a blue circle with a plus sign inside. Click on it to add a device. You will need to provide the (see the live main dashboard for example values for each):

  1. DNS Name
  2. Description (example: Digital Sign)
  3. Area (manages the device: IT Departament)
  4. Location (building and room number)
  5. Department (manages the content on the device)
  6. IP Address
  7. Device Type
  • Removing A Registered Device

    • On the line of the device you want to remove: click the the red cirlce with an X inside to remove the device.

  • Changing the Settings of a Registered Device

    • On the line of the device you want to edit: click the the blue circle with the image of a pencil inside to edit the device settings.

  • Enabling/Disabling Email Notifications for a Registered Device

    • On the line of the device you want to disable the email notifications on: click the toggle bar. When the email notifications are enabled the toggle button will move to the right and the toggle bar bacground will change to green.

  • Temporarily Snoozing Email Notifications on a Registered Device

    • On the line of the device you want to snooze the email notifications on: click the image of a bell. You will be prompted to provide the number of days to snooze the email notifications, and the image will change so the bell now shows includes Z z to indicate it is snoozed. Hovering over the image will reveal how long the device is snoozed. This is useful for when you are performing planned maintenance, or troubleshooting an issue with the device.

StatusChecker

End-user Documentation

  • Status Checker

    • The Status Checker application attempts to ping devices at five minute intervals and if the device does not respond after five minmutes, an email notification is sent out: you have the option to disable notifications for any device or to snooze email notifications: handy for when you are working on, updating, or replacing a device. If a device is shown as not responding you can initiate a ping from the application. There is an SSH button available for each device so you can remote into the device (useful if you are not using any sort of endpoint management solution such as Pi Connect. You can add or remove devices from the Status Checker application from within the application.

  • Inserting Status Badges in your HTML

    • If you want to add a Status Badge to a website using Status Checker - it simple to do. The device or server that you want create a badge for DOES NOT have to be registered with Status Checker application.
